Docker

Docker

輕量級虛擬化技術，容器化技術(Container)

Docker 三元素

  1. Image:映像檔 => 鏡像(映像檔)是一個只讀的模板，用來創建容器
  2. Container:容器 => 容器是從鏡像(映像檔)創建的可執行實例，可以被啟動、開始、停止、刪除
  3. Repository:倉庫 => 倉庫是集中存放鏡像(映像檔)文件的場所
  4. 比喻1: 鏡像(映像檔)是類比於類別，容器是類比於物件，倉庫是類比於類別庫
  5. 比喻2: 鏡像(映像檔)是類比於蛋糕模具，容器是類比於從蛋糕模具烤出的蛋糕，倉庫是類比於放模具的收納櫃

windows 安裝 docker

linux 安裝 docker

常用指令

  • docker –version
  • docker images: 查看本地鏡像(映像檔)
  • docker pull microsoft/dotnet:2.1-sdk: 從 docker hub 下載鏡像
    docker pull microsoft/mssql-server-linux
  • docker run –name myapp -it microsoft/dotnet:2.1-sdk: 執行鏡像(啟動鏡像)
    or
    docker run –name sql_server -m 512m -e ‘ACCEPT_EULA=Y’ -e ‘SA_PASSWORD=yourStrong(!)Password’ -p 11433:1433 -d microsoft/mssql-server-linux
  • docker ps: 查看正在執行的容器
  • docker ps -a: 查看所有容器
  • docker start myapp: 啟動容器
  • docker stop myapp: 停止容器
  • docker restart myapp: 重啟容器
  • docker rm myapp: 刪除容器
  • docker update –restart=always myapp: 設定容器自動重啟
  • 刪除容器前，記得停止容器

Linux 下指令

  • systemctl restart docker: 重啟 docker 服務
